# Java JDBC User CRUD

Projeto desenvolvido em Java com JDBC e PostgreSQL para praticar operações de persistência e consulta em banco de dados relacional.

## Funcionalidades
- Cadastro de usuários
- Listagem de usuários
- Busca de usuário por ID
- Atualização de dados
- Exclusão de usuários
- Cadastro de telefones vinculados a usuários
- Consulta com INNER JOIN
- Exclusão manual de registros pai e filho

## Tecnologias utilizadas
- Java
- Maven
- JDBC
- PostgreSQL

## Conceitos praticados
- DAO Pattern
- PreparedStatement
- Connection
- ResultSet
- Commit e rollback
- CRUD relacional
- Relacionamento entre tabelas

## Estrutura do projeto
- `conexao_jdbc` → conexão com o banco
- `dao` → regras de acesso a dados
- `model` → classes do sistema

## Objetivo do projeto
Este projeto foi desenvolvido com foco em aprendizado prático de Java backend, integração com banco de dados PostgreSQL e manipulação de dados com JDBC.

## Observações
Este projeto utiliza conexão local com PostgreSQL e foi criado como prática de fundamentos de backend em Java.
